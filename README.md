## 🚀 Vercel 배포 방법

### 방법 1: GitHub 연결 (추천)
1. 이 폴더를 GitHub에 push
2. vercel.com 접속
3. "Import Project" 클릭
4. GitHub repo 선택 → 자동 배포

### 방법 2: Vercel CLI
```bash
npm i -g vercel
cd new-smile-dental
vercel
```

### 방법 3: Drag & Drop
1. vercel.com 접속
2. 폴더를 드래그앤드롭
3. 완료!
